
# SnippetVault

SnippetVault is a web application that allows developers to store, manage, and share reusable code snippets.

## Features

- User authentication (Signup/Login)
- Personal dashboard to manage snippets
- Create, edit, and delete snippets
- Public snippets page
- Copy code to clipboard
- Share snippet links

## Tech Stack

- Next.js
- Supabase (Authentication + Database)
- Tailwind CSS
- Vercel (Deployment)

## Project Structure

app/
  login/
  signup/
  dashboard/
  snippets/

  supabase.js
